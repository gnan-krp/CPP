# C++ Practice Programs

This repository contains **10 moderate-level C++ programs** covering important topics like arrays, strings, recursion, functions, structures, pointers, and patterns.

## Programs List

1. **1D_Array_Frequency**: Count frequency of each element in a 1D array.
2. **2D_Array_MatrixMultiplication**: Multiply two matrices entered by the user.
3. **String_Palindrome**: Check if a string is a palindrome ignoring spaces and case.
4. **Recursion_SumOfDigits**: Find the sum of digits of a number using recursion.
5. **Function_SwapNumbers**: Swap two numbers using a function with pointers.
6. **Structure_StudentAverage**: Calculate average marks of a student using a structure.
7. **PointerToStructure_Point**: Modify structure members using a pointer to structure.
8. **PointerToFunction_Calculator**: Simple calculator using function pointers.
9. **Pattern_Diamond**: Print a diamond pattern of stars.
10. **Unique_MatrixTranspose**: Find the transpose of a 3x3 matrix using a function.

## How to Run

1. Open terminal or an IDE like VSCode/CLion.
2. Navigate to the folder of the program you want to run.
3. Compile the program:
    g++ main.cpp -o program
4. Run the compiled program:
    ./program (Linux/macOS)
      program.exe (Windows)
